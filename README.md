# Portfólio Profissional | Otávio Branco

Este projeto é um portfólio profissional desenvolvido para **Otávio Branco**, Consultor de Vendas e Revendedor de Insumos Agrícolas da **Solo Rico**. O site foi projetado para transmitir profissionalismo, confiança e expertise técnica no setor de agronegócio.

## 🚀 Funcionalidades

- **Identidade Visual Solo Rico**: Cores (Verde e Laranja) e tipografia alinhadas à marca oficial.
- **Showcase de Produtos**: Destaque para as linhas Titanium Foliar, Titanium Sollus e Adjuvantes.
- **Conversão Direta**: Botão flutuante e chamadas de ação (CTA) para contato via WhatsApp.
- **Design Responsivo**: Experiência otimizada para Desktop, Tablets e Mobile.
- **Animações Suaves**: Interações fluidas para uma navegação premium.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e SEO.
- **CSS3 (Vanilla)**: Design moderno com Flexbox e Grid.
- **JavaScript (Vanilla)**: Interatividade e animações de scroll.
- **Font Awesome**: Ícones profissionais para interfaces.
- **Google Fonts (Inter)**: Tipografia legível e sofisticada.

## 📂 Estrutura do Projeto

- `index.html`: Estrutura principal do site.
- `index.css`: Estilização completa e responsividade.
- `script.js`: Lógica de scroll suave e animações.
- `Image/`: Diretório contendo a foto oficial de Otávio Branco.
