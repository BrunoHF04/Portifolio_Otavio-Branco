# Portfólio Masterclass Elite | Otávio Branco (Solo Rico)

Este projeto é um portfólio de alta performance e design de luxo desenvolvido para **Otávio Branco**, Consultor de Vendas e Revendedor de Insumos Agrícolas da **Solo Rico**. O site foi elevado ao nível **Masterclass Elite (125%)**, focando em autoridade, conversão e tecnologia de ponta para o setor do agronegócio.

## 🚀 Funcionalidades Premium (Elite UX)

- **📲 PWA (Instalar como App)**: O site funciona como um Aplicativo nativo, instalável na tela inicial do celular com ícone personalizado.
- **✨ Splash Screen Animada**: Carregamento inicial de luxo com animação da marca Solo Rico.
- **🧲 Botões Magnéticos (CTAs)**: Interatividade tátil que atrai o cursor para os pontos de conversão.
- **🌱 Cursor Agro (Broto)**: Substituição do cursor padrão por um ícone customizado de broto, reativo a elementos de interface.
- **✍️ Assinatura Digital Animada**: Efeito de escrita manual que revela a assinatura de Otávio Branco via scroll.
- **📜 Lenis Smooth Scroll**: Rolagem de inércia ultra-fluida para uma navegação premium.
- **📊 Barra de Progresso "Safra"**: Indicador visual no topo que acompanha o avanço da leitura.
- **🤝 Vitrine de Parceiros**: Seção dedicada para exibir logotipos de marcas como Solo Rico, Sollus e Titanium.
- **🔝 Botão Voltar ao Topo**: Botão dinâmico orbital para navegação facilitada em longas sessões.
- **🌓 Modern Dark Mode**: Tema "Deep Forest" otimizado para ambientes de baixa luz e conforto visual.
- **📇 Gerador de VCard**: Botão configurado para salvar o contato direto na agenda do celular do produtor rural.
- **📧 Web3Forms Integration**: Formulário de contato profissional com envio direto para o e-mail sem recarregar a página.

## 🛠️ Tecnologias Utilizadas

- **HTML5 Profissional**: Estrutura semântica focada em SEO e acessibilidade.
- **CSS3 Masterclass**: Uso intensivo de Glassmorphism (efeito vidro), Flexbox, Grid e animações @keyframes complexas.
- **JavaScript (Vanilla)**: Lógica de interatividade, observadores de scroll e manipulação de DOM.
- **Lenis.js**: Motor de scroll suave inercial.
- **Font Awesome**: Conjunto de ícones profissionais.
- **Service Workers & Manifest.json**: Infraestrutura para Progressive Web App (PWA).

## 📂 Estrutura do Projeto

- `index.html`: Estrutura principal do site.
- `index.css`: Design system completo e responsividade Masterclass.
- `script.js`: Lógica de interatividade Elite (Cursor, Botões Magnéticos, Scroll).
- `manifest.json`: Configurações de instalação do Aplicativo.
- `sw.js`: Service worker para cache e funcionalidade offline básica.
- `Image/`: Assets de imagem, incluindo foto oficial, banner rural e ícones PWA.
